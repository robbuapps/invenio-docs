# Positus Messenger

![Positus Messenger Overview](/images/positus/messenger/positus-messenger-visao-geral.png)

## Key Features

💛 WhatsApp Business API—prevent your number from being blocked.\
💙 Multi-user simultaneous customer support.\
🧡 Organize conversations by user with the "follow conversation" option.\
💜 Multiple WhatsApp numbers on the same computer.\
🤎 Tags - Custom labels.\
💚 Advanced chat filtering by agent, tags, and content.\
🖤 Send HSM - Active message templates.\
💖 Integration with CRMs or platforms using the official API.\
🤍 Monitor your phone number's quality rating.\
🤖 Possibility of integration with Robbu chatbot.\

## Add New Contact

Add new contacts and send active [template messages](/docs/positus/messenger/positus-messenger#enviar-mensagem-de-template) to initiate a conversation. Click on the top menu "Add new contact" and input the name, nickname, and phone number including the country code.

Remember that on the WhatsApp Business API, you can only chat with a contact freely if they reach out to you first or if they reply to your template message.

📌 Positus Messenger automatically validates whether the WhatsApp number is valid before saving a new contact.

![Registering a new contact in Positus Messenger](/images/positus/messenger/cadastrar-novo-contato.gif)

<Info>
  Upon receiving new messages, the contact's name will automatically sync with
  their WhatsApp profile name. If you prefer to keep a fixed name, use the
  nickname field right below.
</Info>

## Send Template Messages

WhatsApp templates, also known as message templates, are reusable formats for common messages that a business can send through our Positus Messenger platform. \
\
Every template must be approved by WhatsApp before it can be used—this ensures you are following the allowed content guidelines.

Before submitting your template for approval, remember that it must contain the entire message you wish to send, using placeholders like Name or Address, which will be replaced by each customer's data during delivery.

![Sending an HSM template message](/images/positus/messenger/enviar-template-hsm.gif)

<Info>
  Sending a template message does not open the 24-hour window. The window will
  only open when the recipient replies to your message. \ [Learn
  more.](/docs/positus/messenger/positus-messenger#o-que-é-a-janela-de-24-horas)
</Info>

## What is the 24-Hour Window?

If a user initiates a conversation with you, you can continue it over WhatsApp for up to 24 hours after their last message for free (“24-Hour Window”). Outside of this 24-Hour Window, you can only send messages using approved Message Templates (HSM), for which the applicable rate will be charged.

<Info>
  To learn more, access the **WhatsApp Business Policy**:
  [https://www.whatsapp.com/legal/business-policy/](https://www.whatsapp.com/legal/business-policy/)
</Info>

## What are Tags (Labels)?

Positus Messenger allows you to add tags to chats to manage conversations. Tags or labels are useful for businesses and professionals who want to keep messages from customers, suppliers, and internal topics intelligently organized and listed within the messenger. This feature adds colored flag icons over a chat's description in the chat list.

This way, the user managing contacts receives a visual aid to quickly recognize a topic and gather conversations from each category into a single list.

### Add and Remove Tags from a Conversation

To add a tag, open a conversation and click on the tag indicator icon, then select the desired tags to label and organize the chat. Clicking on a tag that is already selected for that conversation will automatically remove it.

![Adding and removing tags from a conversation](/images/positus/messenger/adicionar-remover-marcadores.gif)

<Info>There is no limit on the number of tags per conversation.</Info>

### Filter Conversations by Tags

Tag Filtering allows you to organize your conversations by segmentation. \
You can filter your chats using the "**Contact with specific tag**" filter and then select the tag you wish to filter by.

![Filtering conversations by tag](/images/positus/messenger/filtro-por-marcadores.gif)

### Manage Tags

Tags can be accessed directly within a conversation in Positus Messenger or through Positus Studio by navigating to "Resources" >> "Numbers" >> (Click on the number you want to create tags for) >> "Tags" >> "Register Tag".

![Managing tags in Positus Studio](/images/positus/messenger/gerenciar-marcadores.gif)

📌 Currently, there is no limit on the number of tags.
📌 Tags are created per phone number, meaning you can have multiple active numbers with different tags.

<Info>
  Only users with the "owner" profile for both the number and workspace can
  manage tags. To learn more about access profiles, see this link: [Click
  Here](/docs/positus/studio/studio-positus#como-convidar-um-novo-usuário-para-o-meu-workspace)
</Info>

## Advanced Conversation Filters

- **All contacts**
  - The list will display **all contacts** available for this number, sorted by the last message received.
- **My contacts**
  - The list will display all contacts followed by the logged-in user, making it easy to see which contacts they are currently assisting.
- **Contacts with no followers**
  - The list will show all contacts without any followers, making it easy to identify which clients are not yet being assisted by any user.
- **Contact with specific follower**
  - Selecting this option will bring up a list of other users associated with this number. After selecting a user, the list will display all contacts being followed by the filtered user.
- **Contact with specific tag**
  - Selecting this option will show a list of tags associated with this number. After selecting a tag, the list will display all contacts marked with that specific filter.

## How to Format Your Messages

WhatsApp allows you to format text inside your messages. This feature cannot be turned off.

#### Italic

To _italicize_ your message, place an underscore before and after the text:

\_text\_

#### Bold

To **bold** your message, place an asterisk before and after the text:

\*text\*

#### Strikethrough

To strikethrough your message, place a tilde before and after the text:

\~text\~

#### Monospace

To monospace your message, place three backticks before and after the text:

\`\`\`text\`\`\`

## How to Delete Sent Messages

It is not possible to delete sent messages using Positus Messenger due to an operational limitation of the WhatsApp Business API. This safeguard ensures your company maintains a more secure environment, preventing inappropriate messages from being sent and subsequently erased.

## Multiple WhatsApp Numbers for the Same User

Your company can have multiple numbers, allowing the same user to switch views within Positus Messenger without needing to reconnect.

_To do this, click on your initials in the top-left menu and select the desired number. This feature will be available if you have more than 1 active number on the platform._
