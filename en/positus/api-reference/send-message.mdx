---
title: Send Message
description: Endpoint for sending messages via WhatsApp
api: "POST https://api.posit.us/v1/messages"
---

## Body

<ParamField body="to" type="string" required>
  Recipient number in international format. E.g.: `5511999999999`
</ParamField>

<ParamField body="type" type="string" required>
  Message type. Values: `text`, `image`, `document`
</ParamField>

<ParamField body="text" type="object">
  Required when `type` is `text`.
  <Expandable title="properties">
    <ParamField body="text.body" type="string" required>
      Message content (max 4096 characters)
    </ParamField>
  </Expandable>
</ParamField>

## Example

```bash
curl -X POST https://api.posit.us/v1/messages \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "5511999999999",
    "type": "text",
    "text": {
      "body": "Hello! Your message via Positus."
    }
  }'
```

<ResponseExample>
```json 200
{
  "id": "msg_abc123",
  "status": "queued",
  "to": "5511999999999"
}
```
</ResponseExample>
