## Bearer Token

Include the token in all requests:

```bash
curl -X GET https://api.posit.us/v1/account \
  -H "Authorization: Bearer YOUR_TOKEN"
```

<ResponseExample>
```json Success
{
  "id": "acc_123",
  "name": "My Company",
  "plan": "pro"
}
```

```json Error
{
  "error": "unauthorized",
  "message": "Invalid or expired token"
}
```
</ResponseExample>
