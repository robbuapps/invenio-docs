# Observações

- Para clientes que utilizam **múltiplos serviços**, basta liberar a **lista combinada dos IPs** de cada grupo acima.  
- Em serviços que compartilham IPs (como InvenioIdr V1, OutputStream e Jobs), a liberação é **única** para todos eles.  
- Em caso de dúvida, a recomendação é **liberar todos os IPs listados nesta documentação**, garantindo total compatibilidade com a plataforma Robbu.  
- Esta lista pode ser atualizada conforme a evolução da infraestrutura.
