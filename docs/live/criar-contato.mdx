## Permissão necessária  
Para adicionar um novo contato, você precisa da permissão adequada. Esta permissão pode ser habilitada no **Invenio Center** seguindo o caminho:  
`Usuários >> Habilitar a opção “Adicionar Contato”`.

## Passos para criar um novo contato  

1. Faça login no **Invenio Live**.  
2. Na tela principal, clique no botão **"Novo Contato"**.  
3. Preencha os campos obrigatórios e selecione o **segmento** correspondente.  
4. Clique em **"Cadastrar"**.  

Após o cadastro, você poderá adicionar um ou mais **canais de comunicação** ao contato.

## Problemas comuns  

### O contato cadastrado não aparece na minha lista  
- Certifique-se de que você é o **usuário responsável** pelo contato.  
- Caso contrário, ele não será exibido na sua lista de contatos.  

### O contato foi cadastrado duplicado, o que fazer?  
- Antes de cadastrar um contato, utilize o **campo de busca** para verificar se ele já existe.  
- O sistema pode **unificar automaticamente contatos duplicados**, caso essa opção esteja habilitada no **Invenio Center**.
