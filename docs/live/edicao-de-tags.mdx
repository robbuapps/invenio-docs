As **tags** (ou **hashtags**) são formas de marcar contatos para que possam ser localizados rapidamente quando necessário.

---

##  Adicionando tags no Invenio Live

1. Acesse um contato no **Invenio Live**.
2. Clique no ícone de **hashtag** para definir uma ou editar uma tag, após a inserção selecione **OK**.
![Adicionando Tag no LIVE](https://robbublob.blob.core.windows.net/docs/invenio-live/Tags1imgLive.png)
![Adicionando Tag no LIVE](https://robbublob.blob.core.windows.net/docs/invenio-live/Tags2imgLive.png)
---

##  Pesquisando por tags no Invenio Live

Utilize o **filtro de busca** para localizar contatos com uma tag específica.

![Filtro por TAG no LIVE](https://robbublob.blob.core.windows.net/docs/pesqtag.png)

---

##  Editando tags no Invenio Center

1. No menu lateral, vá em **Contatos**.

[Clique aqui para acessar diretamente a aba de contatos](https://inveniocenter.robbu.global/painel/contatos)

2. Acesse um contato e clique em **Editar** para modificar ou adicionar uma hashtag.

![Editando TAG no CENTER](https://robbublob.blob.core.windows.net/docs/invenio-live/Tags4ImgCenterEdit4.png)

---

##  Pesquisando por tags no Invenio Center

Use o filtro no menu **Contatos** para buscar contatos com uma tag específica.

![Filtro por TAG no CENTER](https://robbublob.blob.core.windows.net/docs/invenio-live/Tags3ImgCenter.png)

---

>  ⚠️**Importante:**  
> Apenas a **tag atual** do contato pode ser utilizada na busca.  
> **Tags anteriores não são armazenadas** pelo sistema.

---

## ⁉️ Perguntas Frequentes (FAQ)

<AccordionGroup>
  <Accordion title="O que são tags ou hashtags?">
    Tags são marcadores usados para classificar e identificar contatos, facilitando buscas e segmentações dentro do sistema.
  </Accordion>

  <Accordion title="Onde posso adicionar uma tag?">
    Você pode adicionar uma tag tanto no Invenio Live, durante o atendimento, quanto no Invenio Center, ao editar o cadastro do contato.
  </Accordion>

  <Accordion title="Consigo pesquisar por tags antigas de um contato?">
    Não. O sistema apenas reconhece a **tag atual** do contato. Tags anteriores são sobrescritas e não ficam salvas.
  </Accordion>

  <Accordion title="Posso usar mais de uma tag por contato?">
    Atualmente, o sistema permite apenas **uma tag por contato**. Use palavras-chave relevantes para facilitar a categorização.
  </Accordion>

  <Accordion title="Tags são visíveis para todos os usuários?">
    Sim. As tags são compartilhadas entre os usuários que têm acesso aos contatos no Live e no Center.
  </Accordion>
</AccordionGroup>
