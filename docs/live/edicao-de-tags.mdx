As **tags** (ou **hashtags**) são marcadores que permitem classificar contatos para localização rápida e segmentação estratégica dentro do sistema.

<Warning>
**Limitação:** O sistema armazena apenas a **tag atual** do contato. Tags anteriores são sobrescritas e não ficam disponíveis para busca.
</Warning>

---

## Adicionando Tags no Invenio Live

<Steps>
  <Step title="Acessar o contato">
    Abra a conversa com o contato desejado no **Invenio Live**.
  </Step>
  
  <Step title="Clicar no ícone de hashtag">
    Localize e clique no **ícone de hashtag (#)** na interface do contato.
    
    **Caminho:** **Invenio Live > Abrir Conversa > Ícone de Hashtag (#)**
    
    <Frame caption="Ícone de hashtag para adicionar ou editar tags do contato">
      <img src="https://robbublob.blob.core.windows.net/docs/invenio-live/Tags1imgLive.png" />
    </Frame>
  </Step>
  
  <Step title="Inserir ou editar a tag">
    No campo exibido, digite a tag desejada e clique em **"OK"** para salvar.
    
    <Frame caption="Campo de entrada de tag com botão de confirmação">
      <img src="https://robbublob.blob.core.windows.net/docs/invenio-live/Tags2imgLive.png" />
    </Frame>
    
    <Tip>
    Use palavras-chave relevantes e padronizadas para facilitar buscas futuras (ex: "urgente", "vip", "follow-up").
    </Tip>
  </Step>
</Steps>

---

## Pesquisando por Tags no Invenio Live

Para localizar contatos com uma tag específica, utilize o **filtro de busca** na lista de contatos.

**Caminho:** **Invenio Live > Lista de Contatos > Filtro de Busca > Inserir tag desejada**

<Frame caption="Filtro de busca por tags na lista de contatos do Invenio Live">
  <img src="https://robbublob.blob.core.windows.net/docs/pesqtag.png" />
</Frame>

## Editando Tags no Invenio Center

<Steps>
  <Step title="Acessar lista de contatos">
    No **Invenio Center**, clique em **"Contatos"** no menu lateral.
    
    <Tip>
    Acesso direto: [Lista de Contatos no Invenio Center](https://inveniocenter.robbu.global/painel/contatos)
    </Tip>
  </Step>
  
  <Step title="Editar o contato">
    Localize o contato desejado na lista e clique em **"Editar"**.
    
    **Caminho:** **Invenio Center > Contatos > Selecionar contato > Clicar "Editar"**
  </Step>
  
  <Step title="Modificar a tag">
    No formulário de edição, localize o campo **"Tags"** e insira ou modifique a hashtag desejada.
    
    <Frame caption="Campo de edição de tags no formulário do contato no Invenio Center">
      <img src="https://robbublob.blob.core.windows.net/docs/invenio-live/Tags4ImgCenterEdit4.png" />
    </Frame>
  </Step>
  
  <Step title="Salvar alterações">
    Clique em **"Salvar"** para confirmar a modificação.
  </Step>
</Steps>

---

## Pesquisando por Tags no Invenio Center

Utilize o **filtro de busca** no menu de **Contatos** para localizar todos os contatos marcados com uma tag específica.

**Caminho:** **Invenio Center > Contatos > Campo de filtro > Inserir tag**

## ⁉️ Perguntas Frequentes (FAQ)

<AccordionGroup>
  <Accordion title="O que são tags ou hashtags?">
    **Tags** são marcadores textuais usados para **classificar e identificar contatos**, facilitando buscas e segmentações dentro do Invenio Live e Center.
    
    Exemplos de uso:
    - Categorização por status: `#urgente`, `#pendente`, `#follow-up`
    - Segmentação comercial: `#vip`, `#prospect`, `#cliente-ativo`
    - Identificação de contexto: `#reclamacao`, `#elogio`, `#duvida`
  </Accordion>

  <Accordion title="Onde posso adicionar uma tag?">
    Você pode adicionar ou editar tags em **dois locais**:
    
    **1. Invenio Live (durante atendimento):**
    - **Caminho:** Invenio Live > Abrir Conversa > Ícone de Hashtag (#) > Inserir tag > OK
    
    **2. Invenio Center (edição de cadastro):**
    - **Caminho:** Invenio Center > Contatos > Selecionar contato > Editar > Campo "Tags" > Salvar
  </Accordion>

  <Accordion title="Consigo pesquisar por tags antigas de um contato?">
    **Não.** O sistema armazena e permite buscar apenas a **tag atual** do contato.
    
    Quando você altera a tag de um contato, a tag anterior é **sobrescrita** e não fica disponível no histórico. Isso vale para buscas tanto no Live quanto no Center.
  </Accordion>

  <Accordion title="Posso usar mais de uma tag por contato?">
    **Não.** Atualmente, o sistema permite apenas **uma tag por contato**.
    
    <Tip>
    **Dica:** Use palavras-chave compostas ou códigos padronizados para maximizar a utilidade (ex: `#vip-urgente` ou `#prospect-2024`).
    </Tip>
  </Accordion>

  <Accordion title="Tags são visíveis para todos os usuários?">
    **Sim.** As tags são **compartilhadas entre todos os usuários** que têm acesso aos contatos, tanto no Invenio Live quanto no Invenio Center.
    
    Isso facilita a comunicação interna e garante consistência na organização dos contatos entre equipes.
  </Accordion>
</AccordionGroup>
