## Como adicionar um canal de SMS em um contato

Siga o passo a passo abaixo para adicionar um canal de SMS a um contato dentro do Invenio Live:

1. Acesse o **Invenio Live** com seu login.

2. **Abra o contato** desejado.

3. No topo da tela do contato, clique no menu **“Adicionar canal”**.

4. Na tela de **Manutenção de canais**, clique em:
   - **“Adicionar canal”**, caso o contato ainda não possua nenhum canal cadastrado.
   - Ou em **“Editar canais”**, se o contato já possui canais registrados.

5. Na lista de opções, selecione o canal **“SMS”**.

6. Preencha os campos:
   - **Número de telefone**: Insira o número que deseja adicionar.
   - **Responsável**: (Opcional) Informe o nome do responsável por este número. Esse campo é útil quando o número pertence a terceiros relacionados ao contato, como pai, mãe, filho, etc.

7. Clique em **“Salvar”** para concluir.

> 📌 **Importante:** Ao adicionar um novo canal, ele será definido automaticamente como **canal principal** do contato. Ou seja, todas as mensagens enviadas a partir deste momento serão direcionadas a este número por padrão.
