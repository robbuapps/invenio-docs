## Acessando seu perfil
Para acessar as configurações do seu perfil, siga os passos abaixo:

1. Faça login no **Invenio Live**.
2. No menu superior, clique no ícone do usuário para exibir o menu do usuário.
3. Selecione a opção **"Meu Perfil"**.

## Como altero meu avatar (foto)?
Você pode carregar uma nova imagem do seu computador ou remover a imagem atual do seu avatar.

Caso você ainda não tenha um avatar carregado e tenha vinculado sua conta do Instagram (seguindo o formato **@seu_perfil**), tentaremos capturar automaticamente sua foto de perfil do Instagram.

### Passos para alterar o avatar
1. Acesse **Meu Perfil** no menu do usuário.
2. Clique na opção **Alterar Avatar**.
3. Escolha uma nova imagem do seu computador ou remova a imagem atual.
4. Clique em **"Salvar alterações"** para confirmar a mudança.

## Como altero minha senha?
Para alterar sua senha, siga os passos abaixo:

1. Faça login no **Invenio Live**.
2. No menu superior, clique no ícone do usuário para exibir o menu do usuário.
3. Selecione a opção **"Meu Perfil"**.
4. Clique na opção **"Alterar minha senha"**.
5. Digite sua senha atual e a nova senha desejada.
6. Clique em **"Salvar"** para confirmar a alteração.

Se você esquecer sua senha, utilize a opção **"Esqueci minha senha"** na tela de login para redefini-la.
