## Como realizar uma busca

### Para localizar um contato no Invenio Center:
- 1.	Acesse a área de **contatos**.
- 2.	Preencha um ou mais campos no **painel de filtros** com as informações desejadas — como nome, e-mail, CPF, telefone, entre outros.
- 3.	Clique no botão **Pesquisar** para exibir os resultados.
- 4.	Caso deseje limpar todos os campos e iniciar uma nova busca, utilize o botão **Remover filtros**.

## Campos disponíveis para filtragem
### Abaixo estão os principais campos disponíveis no painel de filtros e como cada um pode ser utilizado:
-	**Nome:** Permite buscar contatos pelo **nome completo** ou **razão social**.
-	**E-mail:** Filtra os contatos com base no endereço de **e-mail cadastrado**.
-	**CPF:** Localiza contatos utilizando o número do **CPF**ou **CNPJ**
-	**Código:** Busca pelo **código do cliente**, um identificador único atribuído ao contato.
-	**Protocolo:** Permite localizar contatos com base no **número de protocolo**, vinculado a atendimentos anteriores.
-	**Telefone:** Encontra o contato por meio do **número de telefone** registrado.
-	**Tags:** Filtra contatos com base em **etiquetas (hashtags)** cadastradas, agrupando-os por temas, campanhas ou categorias.
-	**Segmento específico:** Refina a busca para exibir apenas os contatos pertencentes a uma **carteira (segmento)** específica.
-	**Responsável:** Apresenta os contatos vinculados a um **atendente** ou **operador designado**.

## ⁉️ Perguntas Frequentes

<AccordionGroup>


  <Accordion title="Quais campos estão disponíveis para filtragem?">
    Os principais campos disponíveis no painel são:
    - **Nome:** Busca por nome completo ou razão social  
    - **E-mail:** Busca por e-mail cadastrado  
    - **CPF/CNPJ:** Busca por CPF ou CNPJ  
    - **Código:** Busca pelo identificador único do contato  
    - **Protocolo:** Filtra por número de protocolo de atendimentos anteriores  
    - **Telefone:** Localiza pelo número de telefone registrado  
    - **Tags:** Agrupamento por etiquetas temáticas, campanhas ou categorias  
    - **Segmento específico:** Filtra contatos de uma carteira específica  
    - **Responsável:** Exibe contatos vinculados a um atendente específico
  </Accordion>

  <Accordion title="Posso combinar diferentes filtros em uma mesma busca?">
    Sim. O painel de filtros foi projetado para aceitar múltiplos critérios ao mesmo tempo, permitindo buscas mais refinadas e precisas.
  </Accordion>

  <Accordion title="O que acontece se eu não preencher nenhum filtro antes de pesquisar?">
    Ao clicar em **Pesquisar** sem preencher filtros, o sistema pode exibir uma lista mais ampla de contatos ou retornar uma mensagem solicitando que ao menos um critério seja informado, dependendo das configurações do ambiente.
  </Accordion>

  <Accordion title="Qual a finalidade do botão 'Remover filtros'?">
    Esse botão limpa todos os campos preenchidos no painel, permitindo que o usuário reinicie a busca do zero, de forma rápida e prática.
  </Accordion>

  <Accordion title="Posso usar a busca para localizar contatos vinculados a um atendente específico?">
    Sim. Basta utilizar o campo **Responsável** para filtrar os contatos associados a um atendente ou operador específico.
  </Accordion>

</AccordionGroup>
