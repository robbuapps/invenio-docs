Aprenda como gerenciar os IPs e domínios autorizados para acesso externo no **Invenio Center e Invenio Live**, garantindo mais segurança e conformidade para sua empresa. 
Este guia foi desenvolvido para te orientar no controle de permissões e assegurar que somente conexões confiáveis tenham acesso ao ambiente. 

---

<Tip>
    **Acesso direto:** [Gerenciar Host de Acesso](https://inveniocenter.robbu.com.br/painel/configuracoes/hosts)
    </Tip>

##  Passo a passo para cadastrar IPs e domínios

1. Acesse o **Invenio Center**.
2. Clique sobre o **seu perfil** no canto superior direito da tela.
3. No menu suspenso, selecione **"Configurações"**.

> <img src="https://storage.robbu.global/v2/file/F41EEE2A6C893836FFFFD0B89A783AE0" />

4. Selecione a opção **Gerenciar Hosts de Acesso**.  

> <img src="https://storage.robbu.global/v2/file/F41EEE2A6C893836416374F201CC420E" />
   
5. Adicione os **IPs e domínios** que deseja permitir o acesso externo. 

> <img src="https://storage.robbu.global/v2/file/E405EF9711C4B014BB5585A2654750EA" />

6. Após adicionar, clique em **Criar** para aplicar as alterações.

---

##  Configuração do Acesso Externo

Para garantir que um usuário não acesse o Invenio de fora da rede configurada:

1. Certifique-se de que o IP do usuário está cadastrado em **Gerenciar Hosts de Acesso**.

2. Desabilite a opção **Acesso Externo** nas configurações do usuário:

   - Acesse o Invenio Center, clique sobre o seu perfil no canto superior direito da tela.
    > <img src="https://storage.robbu.global/v2/file/AA4314C89F77494314DC13ADCEB7B45E" />

   
   - Selecione o usuário que deseja configurar e desabilite a permissão de **acesso externo** e salve as alterações.
   > <img src="https://storage.robbu.global/v2/file/7CC3B69003F9BED623F9582628FA7665" />

## ⁉️ Perguntas Frequentes (FAQ)

<AccordionGroup>
  <Accordion title="O que é o 'Gerenciar Hosts de Acesso'?">
    É a funcionalidade que permite definir quais IPs ou domínios têm permissão para acessar o Invenio Center/Live de forma externa.
  </Accordion>

  <Accordion title="Quantos IPs posso cadastrar?">
    Não há um limite específico de IPs ou domínios. No entanto, recomenda-se cadastrar somente os realmente necessários para manter a segurança.
  </Accordion>

  <Accordion title="O que acontece se um IP não estiver cadastrado?">
    Usuários tentando acessar o sistema fora da rede cadastrada terão o acesso **negado**, a menos que tenham o acesso externo habilitado e estejam em um IP permitido.
  </Accordion>

  <Accordion title="Como sei se um usuário pode acessar de fora da empresa?">
    Verifique se:
    - O IP externo está cadastrado.
    - O campo “Acesso Externo” está ativado no cadastro do usuário.
  </Accordion>

  <Accordion title="Por que devo bloquear o acesso externo?">
    Para proteger informações confidenciais e evitar riscos de segurança. A restrição permite controlar quem acessa remotamente a plataforma.
  </Accordion>

  <Accordion title="Essa configuração afeta o acesso via celular?">
    Sim. Se o acesso externo não estiver configurado corretamente, o uso fora da rede corporativa (como dados móveis) pode ser bloqueado.
  </Accordion>

  <Accordion title="Qual a Importância de bloquear o acesso externo?">
  Bloquear o acesso externo é essencial para:
​
- **Segurança**:
Ao restringir o acesso externo, você protege os dados sensíveis da empresa contra acessos não autorizados.
​
- **Controle de Acesso**:
Permite decidir quem pode acessar a plataforma remotamente, aumentando o controle organizacional.
​
- **Conformidade**:
Auxilia no cumprimento de regulamentações de segurança e proteção de dados (LGPD, ISO etc.).
 </Accordion>

</AccordionGroup>
