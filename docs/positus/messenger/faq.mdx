# FAQ

## O meu número poderá ser banido?

👎 **Não**. *A Positus provedora oficial do WhatsApp esta autorizada a fornecer a conexão direta com a* [*API oficial do WhatsApp*](/docs/positus/primeiros-passos/introducao)*.* 

## Eu preciso de um aparelho celular?

 👎 **Não.** *A Positus Messenger está conectado diretamente com o WhatsApp, através de uma API autorizada.*

## **Qualquer empresa pode utilizar o Positus Messenger?**

👍 **Sim.** *Basta adquirir um plano de WhatsApp Business API com a Positus.*

## **Posso contratar mais de um número?**

👍 **Sim.** *Você pode contratar diversos números.*

## **Posso** *ter até quantos atendentes simultâneos?*

Você poderá ter diversos atendentes cadastrados, porém o seu limite de acesso simultâneo será definido com base na quantidade de licenças contratadas. Você pode contratar quantas licenças quiser.

## **O mesmo atendente pode estar associado a mais de um número?**

👍 **Sim.** *O mesmo atendente pode estar associado a diversos números.* 

## *Posso usar o Positus Messenger e API conectada ao mesmo CRM ao mesmo tempo?*

👍 **Sim.** *Não existem restrições técnicas para esta prática.*

## *Já tenho um número ativo, posso usar a Sandbox como ambiente de teste?*

👍 **Sim.** Neste caso quando você ativar a sua Sandbox o Positus Messenger será direcionado automaticamente para o seu ambiente de teste. Para retornar ao padrão, basta destruir a sua sandbox enviando uma mensagem de texto para o número da Positus com o seguinte texto: "!destroy".

## Sou um representante comercial da Positus posso gerenciar as contas dos meus clientes?

👍 **Sim.** Você terá acesso total as contas dos seus clientes pelo nosso painel.

## O que é um workspace?

O Workspace é o seu espaço de trabalho, utilizado para organizar o ambiente da sua empresa dentro da plataforma da Positus. Você poderá convidar usuários para atendimento via Positus Messenger, desenvolvedores e seu time financeiro para fazer parte do seu ambiente. 

## Posso ter mais de um workspace?

👍 **Sim.** Porém, este recurso esta disponível apenas para representantes comerciais.

## O que são recursos?

Recursos é o agrupamento de números de um determinado workspace.

## Como faço para vincular ou remover um atendente a um número de WhatsApp?

Primeiro, você precisa convidar o usuário a fazer parte do seu workspace e ele precisará aceitar o convite que será enviado pelo e-mail.\
\
Na sequência, você pode navegar em Recursos >> clique no número que deseja asssociar o usuário e clique no botão "Associar usuário(s)".\
\
Para remover, selecione o usuário na lista e clique no botão "Remover".

## O que é API Tokens?

*Este recurso é utilizado por desenvolvedores de sistema.*\
É uma chave secreta que é enviada dentro de uma requisição para um webservice, de forma a identificar e autenticar o usuário que está chamando o serviço. Sugerimos que crie um token por aplicação, facilitando o seu controle das requisições e aumentando a sua segurança.

Você pode revogar o seu token, clicando no botão "Revogar", 

<Warning>
Não compartilhe o seu token com outras pessoas.
</Warning>

## Porque não vejo a foto(avatar) dos meus contatos?

Por questões de privacidade o Facebook / WhatsApp Business API não fornecem o avatar (foto do perfil) dos usuários.

## Posso usar o Positus Messenger e WhatsApp Business API ao mesmo tempo?

👍 Sim. Não existe nenhuma limitação com relação a utilização da API e do Positus Messenger.

## Por quanto tempo o histórico das minhas conversas ficam salvos?

O seu histórico é armazenado por três meses contato do momento do recebimento da mensagem. 

<Info>
**Dica:** Para aumentar este limite entre em contato com o time comercial.
</Info>
