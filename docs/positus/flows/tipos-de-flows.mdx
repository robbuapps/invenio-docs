# Tipos de Flows

Existem dois tipos de flows que você pode criar na Positus: flows estáticos e flows dinâmicos. Entenda as diferenças e quando usar cada um.

## Flows estáticos

Flows estáticos não efetuam nenhuma troca de informações com um endpoint. Eles seguem um fluxo predefinido, sem qualquer integração externa que troque dados em tempo real durante a execução.

Use flows estáticos quando:
- O fluxo é simples e autocontido
- Não há necessidade de integração com sistemas externos
- Os dados exigidos já estão pré-configurados no flow

## Flows dinâmicos

Flows dinâmicos efetuam trocas de informações com um endpoint que precisa ser definido durante a criação ou edição do flow. Isso permite que o flow se adapte e responda a dados dinâmicos em tempo real.

Use flows dinâmicos quando:
- Você precisa recuperar dados de um sistema externo
- O fluxo depende de informações que mudam ou variam por usuário
- A experiência do usuário deve ser personalizada ou condicional

Para saber mais sobre flows dinâmicos, consulte a página [Flows dinâmicos](/docs/positus/flows/flows-dinamicos).
