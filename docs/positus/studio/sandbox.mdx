# Sandbox

## O que é Sandbox?

A Positus disponibiliza através da nossa Sandbox para WhatsApp Business API o acesso completo a todos os recursos utilizando um número demonstrativo, assim você pode testar seu aplicativo em um ambiente de desenvolvimento até realizar a contratação e sua conta ser aprovada.

<Info>
Para começar a utilização do WhatsApp Business API em produção, é necessário realizar a contratação do serviço e aguardar a aprovação da sua conta pelo Facebook. Mas isso não significa que você precisar esperar para começar a construir sua integração.
</Info>

## Como ativar a minha Sandbox?

* Navegue no menu "Sandbox"
* Abra a câmera do seu celular 
* Aponte para o QR-CODE
* O seu WhatsApp será aberto com um código na caixa de texto
* Clique em enviar mensagem
* ✅ Pronto

![Ativando a Sandbox lendo o QR Code com o celular](/images/positus/studio/ativar-sandbox.gif)



## Como eu configuro o webhook na minha sandbox?

Navegue no menu  "Sandbox>> "Webhook" ou [https://studio.posit.us/sandbox/configuracoes](https://studio.posit.us/sandbox/configuracoes), clique no botão "editar webhook", altere a URL e depois clique no botão "Salvar".

![Configurando o webhook da Sandbox](/images/positus/studio/configurar-webhook-sandbox.gif)



## Eu posso utilizar mais de uma Sandbox ao mesmo tempo?

👎 Não.

## Eu posso convidar mais pessoas para a minha Sandbox?

👍 Sim! Atualmente não existe nenhuma limitação. Basta compartilhar o código da sua Sandbox.\
O código da sua caixa esta disponível no menu "Contatos" ou pelo link [https://studio.posit.us/sandbox/contatos](https://studio.posit.us/sandbox/contatos) 

## Eu posso usar o mesmo token na WhatsApp Business API e no Sandbox?

👍 Sim! 

## Como destruir a minha Sandbox?

Para destruir sua Sandbox, envie a mensagem "destroy!" para o número da Positus +551126264234 ou acesse o site [https://studio.posit.us/sandbox/](https://studio.posit.us/sandbox/) e clique no botão "destruir Sandbox".

![Destruindo a Sandbox no Positus Studio](/images/positus/studio/destruir-sandbox.gif)
