### Request

#### Base URL

`https://api.robbu.global/v1`

```json login.json
{
    "Company": "#####",
    "Username": "#####",
    "Password": "#####"
}
```

#### Parâmetros da Requisição

| **Parâmetro** | **Tipo**   | **Obrigatório** | **Descrição**                           | **Exemplo**      |
|---------------|------------|-----------------|-----------------------------------------|------------------|
| `Company`     | `string`   | Sim             | Identificação única da empresa.         | `12345`          |
| `Username`    | `string`   | Sim             | Nome de usuário para autenticação.      | `usuario_teste`  |
| `Password`    | `string`   | Sim             | Senha associada ao usuário.             | `senha_secreta`  |

```json showLineNumbers
{
    "access_token": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9",
    "expire_in": 324684897,
    "refresh_token": "",
    "token_type": "Bearer"
}
```

#### Parâmetros da Resposta

| **Campo**         | **Descrição**                                             | **Exemplo**                      |
|--------------------|---------------------------------------------------------|-----------------------------------|
| `access_token`    | Token JWT utilizado para autenticação nas chamadas da API. | `eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9` |
| `expire_in`       | Tempo de validade do token em segundos.                   | `3600`                           |
| `refresh_token`   | Token utilizado para renovar o `access_token` (se aplicável). | `null`                           |
| `token_type`      | Tipo do token retornado, geralmente `Bearer`.             | `Bearer`                         |
