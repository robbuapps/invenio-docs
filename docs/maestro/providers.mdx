## Brokers (Provedores)
O Maestro suporta vários _brokers_ de terceiros para realizar o envio das mensagens.

Atualmente, os brokers disponíveis são:
- E-mail:
    - RobbuMail
    - SendGrid
- SMS:
    - Classe A
    - Pontal
- WhatsApp:
    - Positus

## Como configurar

1. O provedor fornece um **token de autenticação** (pode ser um **JWT**, **Bearer Token** ou uma **chave fixa**).  
2. Esse **token** deve ser incluído na configuração deste provedor
3. O sistema envia a mensagem para o **endpoint da API do provedor**.
4. O provedor processa e entrega a mensagem ao destinatário.
5. Uma resposta é retornada, indicando o **status do envio** (sucesso, falha, pendente, etc.).

> ⚠️ ATENÇÃO: Se o seu _broker_ exige que uma URL de Webhook seja adicionada para que o Maestro receba as atualizações de status das mensagens, insira a URL a seguir como URL de Webhook:

```url
https://api.common.maestro.robbu.global/v1/messages/events
```
