## Bearer Token

Incluya el token en todas las solicitudes:

```bash
curl -X GET https://api.posit.us/v1/account \
  -H "Authorization: Bearer SU_TOKEN"
```

<ResponseExample>
```json Éxito
{
  "id": "acc_123",
  "name": "Mi Empresa",
  "plan": "pro"
}
```

```json Error
{
  "error": "unauthorized",
  "message": "Token inválido o expirado"
}
```
</ResponseExample>
