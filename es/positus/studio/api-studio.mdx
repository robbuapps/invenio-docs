# API - Studio

<Info>
Se requiere autenticación Bearer Token en todas las llamadas \
Consulta cómo generar tu token [**haz clic aquí**](/docs/positus/studio/studio-positus#como-gerar-um-token-para-utilizar-na-api)
</Info>

## Workspaces

Consulta de los workspaces vinculados a tu token

[https://api.positus.global/v2/workspaces](https://api.positus.global/v2/workspaces)

```

		"id": "564564-5555-33333-bbb3-3423r2cc",
```
