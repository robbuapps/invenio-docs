# Tipos de Flujos

Existen dos tipos de flujos que puedes crear en Positus: flujos estáticos y flujos dinámicos. Entiende las diferencias y cuándo usar cada uno.

## Flujos estáticos

Los flujos estáticos no realizan ningún intercambio de información con un endpoint. Siguen un flujo predefinido, sin ninguna integración externa que intercambie datos en tiempo real durante la ejecución.

Usa flujos estáticos cuando:
- El flujo es simple y autónomo
- No hay necesidad de integración con sistemas externos
- Los datos requeridos ya están preconfigurados en el flujo

## Flujos dinámicos

Los flujos dinámicos realizan intercambios de información con un endpoint que debe definirse durante la creación o edición del flujo. Esto permite que el flujo se adapte y responda a datos dinámicos en tiempo real.

Usa flujos dinámicos cuando:
- Necesitas recuperar datos de un sistema externo
- El flujo depende de información que cambia o varía por usuario
- La experiencia del usuario debe ser personalizada o condicional

Para más información sobre flujos dinámicos, consulta la página [Flujos dinámicos](/es/positus/flows/flujos-dinamicos).
